The Rust Programming Language

What’s Rust?

Rust is an open source systems language with the backing of Mozilla that aims to solve several problems associated with older systems languages in which security was an afterthought.

Why Rust?

  • A safe systems programming language.
    • SEGFAULT’s and memory overflows are a major cause of vulnerabilities in programs written in C/C++
    • Existing solutions for safety are slow and require a separate runtime with a garbage collector.
  • Modern language design (without overhead)
  • Concurrency with no data races

Features

  • Emphasis on safety, control of memory layout and concurrency.
  • Syntax similar to C++
  • Expression based semantics
  • Does not permit null or dangling pointers.
  • Uses RAII instead of Garbage Collection
